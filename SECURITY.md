# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.x | :x: *unstable releases* |

## Reporting a Vulnerability

If you think you have found a security issue, DO NOT open an issue.
You **MUST** submit your issue at https://gitter.im/Spomky/.
